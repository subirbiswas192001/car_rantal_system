# 📖 About the Project

The **Car Rental System** is a console-based Java application developed using **Object-Oriented Programming (OOP)** concepts. The project simulates a car rental management platform where users can rent cars, manage bookings, and perform administrative operations.

This project was developed as an academic mini-project to demonstrate:

* 🚗 Car rental management
* 👤 User management
* 💳 Payment processing simulation
* 🧩 Object-Oriented Programming concepts
* 📂 File and object handling
* 🔄 Java class interactions

# 🚀 Getting Started

## Prerequisites

* Java JDK 8 or above
* IntelliJ IDEA / Eclipse / VS Code

---

## Clone the Repository

```bash
git clone https://github.com/subirbiswas192001/Car-Rental-System-Java.git
cd Car-Rental-System-Java
```

---

## Compile the Project

```bash
javac Car_Rent.java
```

---

## Run the Project

```bash
java Car_Rent
```
